🖼️ Comment ajouter un logo / favicon
Pour remplacer le logo "06" dans la navbar et le favicon de l'onglet, suis ces étapes :
1
Prépare ton image
Crée une image PNG ou SVG carrée (recommandé : 128×128 px ou 512×512 px pour la qualité). Pour le favicon, utilise un fichier .ico, .png ou .svg.
2
Ajoute le favicon dans <head>
Dans le <head> du HTML, remplace ou ajoute cette ligne :
<link rel="icon" href="/logo.png" type="image/png">
3
Remplace le logo dans la navbar
Trouve le bloc .logo-mark et remplace-le par une balise image :
<img src="/logo.png" alt="Logo" style="width:28px;height:28px;border-radius:6px;object-fit:cover">
4
Place le fichier dans le bon dossier
Copie ton image dans le dossier public/ (ou racine du serveur) avec le nom logo.png. Si tu utilises Express.js, assure-toi d'avoir app.use(express.static('public')).
app.use(express.static(path.join(__dirname, 'public')));
